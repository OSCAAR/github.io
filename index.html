oscaar

Open Source Code Differential Photometry for Advancing Amateur Reseach

Download this project as a .zip file Download this project as a tar.gz file

oscaar is a differential photometry toolkit for everyone! Written in Python, this project is meant to serve as an "out-of-the-box" photometry code for amateurs OR a scaffolding on which professional astronomers can develop robust photometry tools. It's built to be as modular and flexible as possible to make it easy to adapt to your data, wherever it came from.

oscaar is currently in active development and may currently break at any time -- the stable v2.0 release is expected early in the summer of 2013.

Supported Operating Systems

  • Mac OS 10.6, 10.7 and 10.8 (with caveats)
  • Windows Vista and later
  • Ubuntu 12.04 and later

Things to download

  • Python 2.7.x
  • NumPy and SciPy: Python modules for scientific computing
  • PyFITS 2.4.0: A Python module for FITS handling
  • matplotlib: A Python module for plotting your results
  • wxPython: A GUI toolkit that allows you to press buttons rather than manipulate the code from the command line (...unless you want to!)

Optional

  • PyEphem 3.7.5: A Python module providing astronomical calculations within Python
